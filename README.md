## 编译步骤

## 编译步骤
``` bash
# 安装依赖
npm install

# 在 localhost:8080 上部署热更新的开发模式服务器 
npm run dev

# 编译生成成品
npm run build

# 编译生成成品，并查看依赖包分析报告
npm run build --report
```

详情见： [介绍和步骤说明](http://vuejs-templates.github.io/webpack/) 和 [vue-loader 文档](http://vuejs.github.io/vue-loader).
